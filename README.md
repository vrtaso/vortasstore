# VORTAS STORE — Сайт дизайн-студии

Сайт дизайн-студии **VORTAS STORE**, задеплоенный через GitHub Pages.

🔗 **[vortas.store](https://vortas.store)**

---

## Структура

```
/
├── index.html          # Главная страница
├── order.html          # Форма заказа
├── style.css           # Стили главной страницы
├── logo.ico            # Фавикон
└── images/             # Изображения услуг
    ├── vexillum.png
    ├── effigy.png
    ├── tenebration.png
    ├── pactlings_pc.png
    └── pactlings_mb.png
```

## Технологии

- Чистый HTML / CSS / JS — без фреймворков
- [Font Awesome 6.6](https://fontawesome.com/) — иконки
- [Google Fonts](https://fonts.google.com/) — Cinzel Decorative, Cinzel, Rajdhani, Jost
- Telegram Bot API — отправка заказов в бот
- Cloudflare — DNS, CDN, защита от ботов

## Деплой

Сайт хостится через **GitHub Pages** с кастомным доменом `vortas.store`.

Любой пуш в ветку `main` автоматически обновляет сайт.
